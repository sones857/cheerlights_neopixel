# ESP8266 NeoPixel Cheerlights

This is an implementation to make Cheerlights with the ESP8266 and a NeoPixel LED Ring.

http://cheerlights.com  

It uses an HTTP request to get the HEX code of the current color.

Used Libraries: 
ESP8266WiFi&lt;br>
Adafruit_NeoPixel 

use pin 1 for data
